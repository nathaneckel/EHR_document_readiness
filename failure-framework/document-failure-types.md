# Document Failure Types
EHR Document Readiness Project

These codes classify the root mechanism of authority document failure.

F1 – Missing Document
The required authority document does not exist.

F2 – Wrong Authority Type
Financial authority required but only medical authority exists, or vice versa.

F3 – Expired / Outdated Document
Document legally invalid due to age or superseded versions.

F4 – Document Inaccessible
Document exists but cannot be located or produced.

F5 – Conflicting Authority
Multiple individuals claim decision authority.

F6 – Jurisdiction Mismatch
Document valid in one state but not recognized by institution or jurisdiction.

F7 – Institutional Rejection
Hospital or facility refuses to accept document despite family expectations.
